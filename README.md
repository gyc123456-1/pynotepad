# 下载
1. 点击右边的 Release
2. 下载你需要的版本
3. 解压
4. 打开文件夹中的 pynotepad.exe
5. 更多帮助请查看 [Wiki](https://github.com/gyc123456-1/pynotepad/wiki)

# 运行源代码
1. 克隆这个仓库
2. 打开命令提示符
3. 输入 `python -m pip install -r requirements.txt` 以安装依赖
4. 运行 main.py 即可
